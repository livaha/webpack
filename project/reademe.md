## webpack+gulp搭建react项目

### 1 安装

安装nodejs最新版：v10.15.3

新建工程文件project

~~~shell

cd project
npm init  #初始化工程
npm i yarn  #使用yarn来安装包

yarn add webpack   webpack-cli 
yarn add gulp gulp-cli


~~~

<https://webpack.js.org/guides/getting-started>   v4.29.6 -->Getting Started
